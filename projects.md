# Projects

A running list of code/app projects built in this workspace.

## Quote Card Generator
- **File:** `quote-card.html`
- **What it does:** Type a quote + author, preview a styled card live, download as PNG or JPG
- **Stack:** Vanilla HTML/CSS/JS, Playfair Display font (Google Fonts CDN), html2canvas CDN
- **Notes:** Self-contained, no build step — open directly in browser

---

## Batch Quote Card Generator
- **File:** `quote-batch.html`
- **What it does:** Paste multiple quotes (`Quote | Author` format), preview all as styled cards, download as ZIP or individual PNGs
- **Stack:** Vanilla HTML/CSS/JS, DM Serif Display + DM Mono (Google Fonts CDN), html2canvas + JSZip + FileSaver CDN
- **Notes:** 5 card colour themes, adjustable font size, per-card hover to download single PNG/JPG

## Landing Page

- **What it does:** Hub page listing all punk projects with links, descriptions, and tags
- **Stack:** Vanilla HTML/CSS, DM Serif Display + DM Mono (Google Fonts CDN)
- **Notes:** Update whenever a new project is added
